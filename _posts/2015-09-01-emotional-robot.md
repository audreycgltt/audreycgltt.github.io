It was at first a simple 3D printed robotic face able to move using six servo motors called Maki, made by [Hello Robo](https://www.hello-robo.com/). The objective was to give it more personality and skills to be able to mimic emotions in order to teach autistic children to develop social abilities. 

For that, we redesigned some 3D parts to include cameras in the eyes, eyebrows and a mobile jaw. 

![Maki 3D model](/assets/img/projects/maki/maki_3D.png)

The controller used was a NI myRio board so all the programming was done using LabVIEW.

This project was continued by other students in following semesters.
